# 備婚小幫手 PWA — 安裝教學

## 📁 檔案說明
```
wedding-pwa/
├── index.html      ← 主程式
├── manifest.json   ← PWA 設定
├── sw.js           ← Service Worker（離線支援）
└── icons/
    ├── icon-192.png
    └── icon-512.png
```

---

## 🚀 方式一：GitHub Pages（免費、推薦）

1. 前往 https://github.com 建立新的 Repository（名稱如 `wedding-app`）
2. 把資料夾內所有檔案上傳到 Repository
3. 進入 Settings → Pages → Source 選 `main` branch
4. 幾分鐘後，你的 APP 會在 `https://你的帳號.github.io/wedding-app` 上線

---

## 🚀 方式二：Netlify Drop（最簡單）

1. 前往 https://app.netlify.com/drop
2. 直接把 `wedding-pwa` 資料夾拖進去
3. 立刻得到一個網址，完成！

---

## 📲 安裝到手機

### iPhone / iPad（Safari）
1. 用 Safari 開啟網址
2. 點底部「分享」按鈕 □↑
3. 選「加入主畫面」
4. 完成！桌面會出現 APP 圖示

### Android（Chrome）
1. 用 Chrome 開啟網址
2. 點右上角「⋮」
3. 選「新增到主畫面」或 APP 會自動提示安裝
4. 完成！
